How to add a theme to the Github Blog?

Feeling the original Github page looks super ugly? This tutorial will teach how to add a concise and user-fridenly theme to the blog!

Below are several good looking themes that you may like,

Theme 1： maupassant
URL: https://kai-wei-626.github.io/
Download link: https://github.com/iissnan/hexo-theme-next

Theme 2： maupassant
URL: https://www.haomwei.com/technology/maupassant-hexo.html
Download link: https://github.com/icylogic/maupassant-hexo

Theme 3： yilia
URL: http://litten.me/
Download link: https://github.com/litten/hexo-theme-yilia

Theme 4： bluelake
URL: https://chaoo.oschina.io/
Download link: https://github.com/chaooo/hexo-theme-BlueLake

  1. Now, open the terminal and go to the blog/themes folder, then type in git clone “url”, which will copy the theme to your blog.

  2. Afterwards, we will see that theme in the themes folder

  1. In the end, open _config.yml and edit the theme value to whatever your theme’s name is.

    theme value has to be exactly same as the name of in folder in the themes folder.

Alright, type in hexo s and check the changes!

How to create Github page on Mac

Posted on 2018-11-17 | Views

How to build a fancy github blog and deploy on the internet?

This tutorial teaches how to create a FANCY github page, which can be used as a blog! Also, I will add how to change your theme in my next blog!

Create a new repository

tips here: make sure the name of repository is same as the owner name!

Go to the repo you just created, click on the ‘setting’ and scroll down so you can open the url below to check your website

Install node.js and git

Use below two links to download,

https://nodejs.org/en/download/

https://git-scm.com/download/

After installation, open the terminal on Mac(open cmd on windows),

  1. Alright, we are half way through this, now, create a folder called blog somewhere on the Mac.

  2. Go to that folder on your terminal by, cd ‘folder structure’, then type in

    npm install hexo -g

  3. After this, type in hexo -v to check whether the installation is completed

  4. Type in hexo init, this might take a minute. You will see ‘start bloggin with Hexo’ when done.

  5. Then type in below command in orders one by one, the local server will start right away

    npm install, hexo g, hexo s

if you have problem installing those packages, try type sudo in front of these commands.

Github/Local association

Now we have set up all the local configuration, we need to associate local environment with github so that we can push updates to github. So here we will use Git that we just installed, first go to the blog folder, and for windows users, right click your mouse and hit git bash here, for mac user, route the terminal to the blog folder.

  1. type in git config –global user.name “your name here” (2 dash)

  2. type in git config –global user.email “your email here”

  3. type in cd ~/.ssh, check if there is .ssh folder, if there is one, you may delete that folder(it depends)

  4. type in ssh-keygen -t rsa -C “your email”, the email should be the same as the one before. There are 3 configs needs to be entered, just hit enter and leave them blank, after that there will be 2 files, id_rsa and id_rsa.pub, which are stored in /.ssh folder.

  5. type in ssh-add ~/.ssh/id_rsa, add the generated SSH Key to ssh-agent,
    go to github website, go to setting, then go to ssh and gpg keys.


create a new ssh key and copy the content in id_res.pub to the box.

  1. type in ssh -T git@github.com check if ssh works.

  2. type in npm install hexo-deployer-git –save to install the last package

  3. It’s almost done here! Last step: go to blog folder and open __config.yml,

    this is the most important config file, make sure you get that correctly.
    the repository value in the config file is the SSH value that copied from github.(see below 2 pics)

Start Blogging

All the configurations have been set up. All your blogs will be saved under /source/_posts as .md.

  1. Add new blog by type in hexo new “blog name”
  2. Edit them in .md files by text editor by using markdown syntax.

More operations please check in helloworld.md file.
